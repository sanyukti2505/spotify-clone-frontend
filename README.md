# 🎵 Mini Spotify Clone
A lightweight, frontend-only Spotify clone built with HTML5, CSS3, and Vanilla JavaScript.

## ✨ Features
- Play/Pause functionality
- Dynamic song switching (Karan Aujla, AP Dhillon, etc.)
- Working Seek Bar to control song progress
- Fully Responsive Design

## 🛠️ Tech Stack
- **HTML5:** Semantic structure
- **CSS3:** Flexbox and Custom Gradients
- **JavaScript:** Audio API for music playback
